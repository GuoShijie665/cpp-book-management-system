# C++ 控制台图书管理系统

## 📋 项目简介
基于 C++17 标准开发的控制台版图书管理系统。项目采用**面向对象（OOP）**设计思想，封装了 Book、User、Library 等核心类，利用 STL 容器与算法实现高效的数据管理，并通过文件流（fstream）完成数据持久化，确保重启程序后数据不丢失。

## 🔧 技术栈
- **语言**：C++17
- **核心特性**：面向对象编程 (OOP)、STL 容器 (vector)、STL 算法、文件 IO 操作、异常处理
- **开发环境**：Windows 10 + Visual Studio

## 🚀 核心功能
1. **用户管理模块**
   - 实现用户注册与登录验证
   - 区分普通用户与管理员权限，精细化控制操作边界

2. **图书管理模块**
   - 支持图书信息的增、删、改、查（CRUD）
   - 实现按编号、书名、作者等多关键字精准检索

3. **借阅管理模块**
   - 模拟真实图书借阅与归还流程
   - 自动更新借阅状态与时间戳，防止超借

4. **数据持久化**
   - 利用文本文件（txt）作为简易数据库
   - 程序启动时自动加载数据，退出时保存状态

## 💡 技术亮点
- **模块化设计**：通过类的封装降低耦合度，代码结构清晰，易于维护。
- **现代 C++ 风格**：深度运用 STL 算法（`std::find_if`、`std::for_each` 等）替代手写循环，提升代码简洁性与效率。
- **健壮性处理**：针对文件打开失败、非法输入、空容器操作等场景做了完善的异常捕获与容错。
